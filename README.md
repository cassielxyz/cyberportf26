## 🔒 About Me

Passionate about networking fundamentals and cybersecurity basics. I am a dedicated lifelong learner focused on building real-world tools that solve practical problems in the digital security space. Beyond the terminal, I am a creative video editor with a passion for high-production visual storytelling.

## ⚡ Key Features

- **🌐 Cyber-Themed UI**: A high-fidelity "Black & Gamma Green-Blue" aesthetic designed for the modern security professional.
- **✨ Animated Hero**: Interactive "SECURING THE NETWORK" text scramble with sleek spring animations.
- **🛠️ Expertise Grid**: Categorized skills covering Networking, Security, Programming, and Creative Design Tools.
- **🎥 Video Showcase**: Cinematic project display with hover-to-play, seamless looping, and a custom unmute toggle.
- **📱 Ultra Responsive**: Fully optimized for every screen size, from mobile to ultra-wide monitors.
- **🚀 Performance-First**: Built with Next.js 15 and Turbopack for near-instant load times and buttery smooth transitions.

## 🛠️ Tech Stack

- **Core**: Next.js 15 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS v4, Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Data**: Static GitHub & WakaTime API Mocks (for stability)

## 📁 Featured Projects

- **PhishShield**: A phishing detection system designed to identify suspicious links and protect users.
- **Malicious-URL-Detector**: A tool to detect malicious URLs using pattern recognition and security analysis.
- **Network Experiments**: Implementations covering core networking concepts and experimental configurations.
- **Drive-uploads**: Efficient file management and upload handling system.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Build for Production

```bash
npm run build
npm run start
```
